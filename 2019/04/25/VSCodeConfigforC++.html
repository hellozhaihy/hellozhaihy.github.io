VSCode C++开发环境配置

Apr 25, 2019

之前尝试过一次在VSCode中搭建C++开发环境，看了一下文档感觉好麻烦，于是转移到VS中写C++代码。最近想用C++刷LeetCode题目，VSCode中刚好有个LeetCode刷题的插件，便又试着在VSCode中搭建C++环境，还好，在官方帮助的指导下整个过程还是挺简单的。

不同的编译器对应不同的开发环境搭建方式，我这里会讲一下使用MinGW和MSVC编译器搭建开发环境。不过，我推荐使用MSVC编译器，因为在实际操作过程中，我发现MinGW编译器不能很好地进行语法检查。比如下面两行代码：

 int cnt =10;
 int bad[cnt];

我先定义了一个整型变量cnt，接着又定义了一个cnt大小的数组。因为cnt不是一个常量，所以这两句是有问题的，但是这在MinGW编译器中检查不出来。

使用MSVC编译器

无论什么编译器，C++ extension for VS Code插件是必须要安装的。MSVS编译器需要通过安装VS和VS中的C++开发包来安装，我电脑上安装的是VS 2019社区版。

必须使用VS shell在桌面新建一个helloworld文件夹，并使用VSCode打开：

 mkdir helloworld
 cd helloworld
 code .

开发环境的配置需要使用到三个json文件：c_cpp_properties.json、tasks.json、launch.json

选择shell CTRL+SHIFT+P 输入：select default shell选择command prompt
