在GitHub建立自己的博客

Feb 02, 2019

第一篇博客，记录一下建立整个博客的过程。

GitHub为每位用户提供了一个免费的域名 https://username.github.io/ ，可以在它上面建立静态网站，也可以建立个人博客。GitHub推荐的博客工具是jekyll。jekyll是一种静态网站生成器，如果你用Markdown写博客，它可以自动地把Markdown文件渲染为网页。结合liquid语法，jekyll可以实现一些高级的功能。jekyll提供了很多的博客模板，让你可以轻松地建立博客。如果你对HTML、CSS和JavaScript熟悉，修改这些模板或者建立自己的页面以达到自己满意的效果也是很容易的。

在这次建立博客的过程中，我主要使用了jekyll、Bootstrap（前端框架）和Git，以下是完整的流程：

安装jekyll、Bootstrap和Git

以Windows 10安装目前最新的jekyll 3.8.5、Bootstrap 4.2.1和Git 2.20.1为例。由于jekyll是用Ruby语言写的，因此安装jekyll需要先安装Ruby。在Ruby下载页面下载安装 Ruby+Devkit 2.6.1-1 (x64).exe（32位系统安装x86版本），然后打开命令行窗口输入 gem install jekyll bundler 安装jekyll。测试jekyll是否安装成功，可以输入 jekyll -V 查看是否能输出版本号。如果能输出版本号，则说明安装成功。

Bootstrap的安装较为简单（如果不使用Bootstrap可以不用安装）。在Bootstrap下载页面下载编译好的的 CSS 和 JS ，然后解压到博客目录，在HTML文件正确引用即可。

<link rel="stylesheet" href="/css/bootstrap.min.css">
<script src="/js/bootstrap.min.js"></script>

Git 的安装也比较简单，在Git下载页面下载安装Git即可。

新建GitHub博客仓库

登录GitHub，新建仓库，命名为yourusername.github.io。使用 https://yourusername.github.io/ 访问的就是该仓库下的文件。

使用jekyll生成博客

命令jekyll new myblog可以生成一个jekyll默认的博客。命令行切换到桌面，输入jekyll new myblog，会发现桌面多了一个myblog文件夹。切换到myblog 文件夹下，再输入 jekyll serve ，打开 http://localhost:4000 。这就是使用 jekyll 建立的第一个博客。
